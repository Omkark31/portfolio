### Local Development Server (Optional)

For a better development experience, you can use a local server:

#### Using Python 3:
```bash
python -m http.server 8000
```

#### Using Python 2:
```bash
python -m SimpleHTTPServer 8000
```

#### Using Node.js (with http-server):
```bash
npx http-server
```

Then open `http://localhost:8000` in your browser.

## ğŸ¯ Customization

### Personal Information

1. **Hero Section**: Edit the hero title, description, and stats in `index.html`
2. **About Section**: Update the about text and highlights
3. **Skills**: Modify the skill categories and tags to match your expertise
4. **Projects**: Replace project cards with your actual projects
5. **Experience**: Update the timeline with your work history
6. **Contact**: Update contact information and social links

### Colors

The color scheme can be customized in `styles.css` by modifying the CSS variables at the top:

```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    --accent-color: #ec4899;
    /* ... more variables */
}
```

### Images

Replace the placeholder image in the About section:
- Add your photo to the project folder
- Update the `.image-placeholder` div in `index.html` with an `<img>` tag

### Projects

To add your projects:
1. Replace project placeholders with actual project images
2. Update project titles, descriptions, and tags
3. Add real links to project demos and GitHub repositories
